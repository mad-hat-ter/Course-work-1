# Руководство Менеджера


## 1. Применение

- Создание листов расписания
- Добавление новых сотрудников на лист "Сотрудники"
- Добавление листов в бота и определение времени открытия записи
- Отслеживание работоспобности бота

---

## 2. Требования

Google аккаунт с правами редактора таблицы смен

---

## 3. Структура Google Таблицы

**Служебные листы (скрытые):**

| Лист | Назначение |
|------|------------|
| Настройки | B1: Базовая ставка, B2: Повышенная ставка, B3: Время N уведомления до смены |
| Сотрудники | Телеграм айди, ФИ, Роль, Активен? |
| Метаданные | Название таблицы, Дата и время открытия доступа к записи, Активно? |
| Журнал | Лог записей на смену (работает автоматически) |

**Листы расписания:** имя `ДД.ММ–ДД.ММ`; строки — часы 09:00–00:00 МСК; столбцы B–H — дни недели.

---

## 4. Инструкция

### Добавление куратора

1. Запросить telegram id нового сотрудника
2. Добавить строку на лист "Сотрудники": ID, ФИ, `curator`, `TRUE`
3. Сотруднику нужно отправить боту `/start`

### Добавление новой недели в бота

1. Создать лист с датами недели в таблице, создать сетку смен
2. В боте нажать "Добавить расписание"
3. Ввести имя листа из таблицы
4. Ввести дату/время открытия. До наступления этого времени кураторы не видят неделю в записи.

### Изменение ставок и уведомлений

Можно отредактировать ячейки B1–B3 листа «Настройки». Перезапуск бота не требуется.

### Роли

| role | Доступ |
|------|--------|
| curator | Запись, личная статистика |
| admin | + добавление расписания |
| accountant | Только отчёт отдела |

---

## 5. Ошибки

| Что произошло | Причина | Устранение |
|---------|---------|------------|
| Нет доступа у сотрудника | Нет строки на листе "Сотрудники" / введен неверный ID | Нужно проверить данные листа «Сотрудники» |
| Неделя не в списке записи | Не добавлена бота зарегистрирована / неверное время открытия | Нужно проверить данные листа «Метаданные» |
| Медленный ответ / ошибки бота | Не работает Redis / нет связи между компонентами | Передача информации программисту |
| Дубли в ячейках | Одновременная запись | Нужно проверить данные листа «Журнал» |
