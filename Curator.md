# Руководство пользователя

## 1. Назначение программы

**Для куратора:** просмотр свободных смен, запись на смену, просмотр графика и оплаты, напоминания перед сменой.

**Для бухгалтера:** формирование отчёта по сменам и начислениям всех кураторов за выбранные недели.

---

## 2. Условия работы программы

### Технические средства

- Устройство с Telegram
- Подключение к интернету

### Условия доступа

1. Учётная запись в Telegram
2. Регистрация администратором на листе «Сотрудники» Google-таблицы

### Запуск

Открыть бота в Telegram, отправить команду `/start`. При успешной авторизации отобразится главное меню с кнопками.

Команды `/menu` и кнопка «Главное меню» возвращают в начальное состояние.

---

## 3. Работа с ботом

### Главное меню

| Кнопка                     | Роль                   |
| -------------------------- | ---------------------- |
| Просмотр свободных смен    | Куратор, администратор |
| Мой график и статистика    | Куратор, администратор |
| Просмотр статистики отдела | Бухгалтер              |
| Главное меню               | Все                    |

### Запись на смену

| Шаг | Кнопка                    | Ответ                     |
| --- | ------------------------- | ------------------------- |
| 1   | "Просмотр свободных смен" | Список недель             |
| 2   | Выбор недели              | Список дней               |
| 3   | Выбор дня                 | Список часов              |
| 4   | Выбор часа                | Запрос на подтверждение   |
| 5   | «✅ Да»                   | Запись в таблицу          |
| —   | «◀ Назад»                 | Возврат на предыдущий шаг |
| —   | «❌ Нет»                  | Отмена записи             |

Записаться можно на смены с датой не раньше начала следующего часа по МСК, нельзя занять занятый слот, нельзя дублировать свою смену на тот же день и час.

### График и статистика

| Шаг | Кнопка                                         | Ответ                                 |
| --- | ---------------------------------------------- | ------------------------------------- |
| 1   | "Мой график и статистика"                      | Список недель                         |
| 2   | Отметка недель, при необходимости «Все недели» | Появление ✅ около выбранных недель   |
| 3   | "Показать отчёт"                               | Отчёт: смены, оплата по неделям, итог |

Учитываются только основные смены (без `*`); первые 14 в неделе — базовая ставка, далее — повышенная. Резервные смены в оплату не входят.

### Статистика отдела

Аналогично графику и статистике, кнопка «Просмотр статистики отдела». Отчёт содержит всех кураторов: ФИ, число смен, сумма начисления, итого.

### Уведомления

За _N_ минут до начала часа смены приходит сообщение: «Привет! Пора на смену».

## 4. Ошибки

| Сообщение                              | Причина                      | Действие                        |
| -------------------------------------- | ---------------------------- | ------------------------------- |
| У вас нет доступа к боту               | ID не в Листе "Сотрудники"   | Обратиться к администратору     |
| Недостаточно прав                      | Неверная роль                | Использовать функции своей роли |
| Нет свободных смен для записи          | Нет открытых недель / слотов | Повторить позже                 |
| Выбранная смена недоступна для записи  | Слот уже занят               | Выбрать другой час              |
| Нет смен в выбранном периоде           | Нет данных на листах         | Выбрать другие недели           |
| Не удалось связаться с Google Таблицей | Сбой связи                   | Повторить через минуту, при повторе ошибки обратиться к администратору          |
| Произошла ошибка. Перезагрузите бота   | Внутренняя ошибка            | Нажать `/menu` или `/start`, при повторе ошибки обратиться к администратору           |
